ZoomWizard

Підбір оптимальної фокусної відстані та типу камери

Про ZoomWizard

ZoomWizard допомагає підібрати оптимальну фокусну відстань об'єктива для конкретних умов спостереження.

Що таке щільність пікселів (px/m)?

Це кількість пікселів, що припадає на 1 метр об'єкта в кадрі. Визначає рівень деталізації:

  • 25 px/m - Детекція (виявлення людини в кадрі)
  • 62 px/m - Огляд (розрізнення деталей одягу)
  • 125 px/m - Розпізнавання (знайомої людини)
  • 250 px/m - Ідентифікація (чітка ідентифікація людини)

Типи камер за фокусною відстанню:

  • Варіофокальна (2.8-12 мм) - для загального огляду та ідентифікації на невеликій відстані
  • Long-range (12-50 мм) - для спостереження на середній відстані
  • PTZ SpeedDome (>50 мм) - для спостереження на великій відстані з можливістю масштабування

Технічні деталі:

  • Більший сенсор дозволяє досягти ширшого кута огляду при тій самій фокусній відстані
  • Оптимальний кут огляду для більшості випадків становить 45-60 градусів
  • Для спостереження за вхідними зонами рекомендується кут 80-90 градусів
  • Для масштабованого спостереження на великій відстані доцільно використовувати PTZ камери
